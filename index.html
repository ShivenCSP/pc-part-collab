Computer Parts

CPU

Also known as the core processing unit, the CPU is the brain of the computer and executes tasks. It has an arithmetic logic unit to perform functions and a register to store instructions. It operates by retrieving instructions from the memory and then executing the. The better the CPU is the better the processing speed is.

CPU companies
  • Intel
  • AMD
  • Qualcomm

GPU

GPU stands for Graphics Processing Unit and is also known as a graphics card. The GPU handles processing the graphics and helping it be shown on the monitor. Without the GPU, your monitor would show nothing.

GPU companies
  • Intel
  • AMD(Ryzen)
  • Nvidia

Motherboard

The motherboard is the main circuit board which acts as the backbone of the computer. It connects together all of the internal parts of the computer and gives them power. It allows all of the internal parts to communicate with each other.

Motherboard Companies
  • Asus
  • MSI
  • Gigabyte

RAM

The RAM is also known as the Random Access Memory. It stores information on the computer that will be needed by the programs currently running for quick access. This information will be erased after the computer is turned off.

RAM Companies
  • Samsung
  • Micron
  • Cosair

ROM

ROM stands for Read-only memory. It stores important instructions for the computer such as how to boot up. It still stays there even after the computer is turned off.

ROM companies
  • Samsung
  • Micron
  • SK Hynix

POWER SUPPLY UNIT

The power supply unit is the main source of power for the PC. It draws a certain amount of power needed to run the PC for however long it is turned on for, it is continously running. They can range from 500 watts to 1600 watts max.

PSU companies
  • Corsair
  • EVGA
  • Gigabyte
